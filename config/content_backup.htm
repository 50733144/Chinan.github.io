Github倉儲管理

我的倉儲: https://50733144.github.io/Chinan.github.io/

指令碼:

從遠端更新倉儲指令碼:

git pull origin master 將遠端的新資料。git fetch 遠端的 branch，然後與本地端的 branch 做 merge，產生一個 merge commit 節點，同步到自己的倉儲。

git pull 將遠端的倉儲直接取代掉原先自己的倉儲。

啟用動態網頁編輯碼:

原先啟用前，須進入cmsimde資料夾，接著打入python wsgi.py才可啟用動態編輯，

如今進入動態前只需，進入自己資料夾，打入cms即可進入動態網頁進行編輯。

推靜態:進入自己的資料夾打入acp即可完成git add ,git commit ,git push
